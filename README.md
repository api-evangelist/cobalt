# Cobalt (cobalt)

Cobalt is a next-generation embedded integration platform (iPaaS) for SaaS products. With more than 1,000 connectors, AI-driven automation, and a rich set of primitives — linked accounts, applications, configurations, events, webhooks, executions, public workflows, datastores, and a Cobalt-hosted portal — Cobalt lets product and engineering teams build native integrations, deploy them within days, and monetize them with the help of AI agents.

#### Features

- **Linked Accounts:** Manage end-customer linked accounts representing each tenant of your SaaS.
- **Applications Catalog:** Browse the 1,000+ third-party applications Cobalt can connect to.
- **Configurations:** Per-linked-account configuration of integration settings and dynamic fields.
- **Events and Webhooks:** Trigger Cobalt events and subscribe to webhooks for real-time notifications.
- **Workflows:** Build, deploy, and execute integration workflows publicly or per-tenant.
- **Executions:** Inspect workflow execution history and logs.
- **Datastores:** Persist integration state with Cobalt-managed datastores.
- **Cobalt Hosted Portal:** Generate hosted portal URLs to embed Cobalt UI for end customers.
- **Session Tokens:** Short-lived tokens to authenticate frontend SDK calls per linked account.

#### Use Cases

- **Embedded Integrations Marketplace:** Offer customers a turnkey integrations marketplace inside your SaaS product.
- **Per-Tenant Workflow Automation:** Run customer-specific automations across CRM, billing, support, and HR systems.
- **AI Integration Agents:** Use Cobalt's AI capabilities to build agentic workflows over connected SaaS apps.
- **Native Connector Replacement:** Replace dozens of point-to-point connectors with one embedded iPaaS.
